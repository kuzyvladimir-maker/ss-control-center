# Бэклог — нереализованные идеи и будущие модули

Это канонический реестр ещё-не-построенных модулей и идей: когда Владимир спрашивает «какие у нас планы и нереализованные идеи», ответ собираем отсюда.

## Идеи

### Отд 6 (Работа с публикой) — модуль привлечения НОВОЙ публики / лидоген
Сейчас есть только Customer hub (работа с существующими обращениями). Нет инструмента для привлечения новой аудитории. Надо продумать каналы привлечения. Источник: оргсхема ЛРХ, отделение 6. Дата: 2026-06-21.

### Отд 5 (Квалификация) — Training / курсы для персонала
Кроме Account Health (комплаенс) нет модуля обучения. Идея: модуль **Training** — курсы для сотрудников + отметка о прохождении («прочитал / сдал»). **Готовые курсы уже есть у Владимира на Google Drive** — их можно оцифровать в модуль (не писать с нуля). Шаг 1: попросить Джеки (OpenClaw / Google Drive) откопать курсы на Drive и составить список. В сайдбаре уже стоит заглушка «Training · Soon». Источник: оргсхема ЛРХ, отделение 5. Дата: 2026-06-21.

### Отд 1 (Коммуникации) — Staff Hats / инструкции / SOP
Нет модуля по сотрудникам и их «шляпам» (описаниям постов), инструкциям, SOP, оргполитике. Это функция ОХС/HCO (отделение 1). Надо создать модуль(и): справочник сотрудников + их шляпы/обязанности + библиотека инструкций/SOP. В сайдбаре уже стоит заглушка «Staff Hats · Soon». Источник: оргсхема ЛРХ, отделение 1. Дата: 2026-06-21.

### Отд 1 (Коммуникации) — Statistics (статистики по отделениям/процессам)
Модуль статистик: по каждому отделению и каждому ключевому процессу — своя стат, которую вводят и графят (еженедельно), а по трендам присваивают «состояния» (conditions). Оргсхема ЛРХ: это Департамент 3 ОХС — «Инспекции и Отчёты» (отделение 1). **Отдельно от Dashboard** (Отд 7): Dashboard — исполнительный пульт, который потребляет эти статистики. Тех состояний/статистик — [05-formuly-sostoyaniy-i-statistiki.md](lrh-green-volumes/05-formuly-sostoyaniy-i-statistiki.md). В сайдбаре заглушка «Statistics · Soon». Дата: 2026-06-21.
