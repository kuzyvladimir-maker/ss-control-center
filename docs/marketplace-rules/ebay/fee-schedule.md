# eBay Marketplace — Fee Schedule

> **Source:** https://www.ebay.com/help/selling/fees-credits-invoices/fees-selling-ebay-business-sellers
> **Last verified:** 2026-05-17
> **Priority:** P2

---

## TL;DR

eBay fees = Insertion Fee (per listing) + Final Value Fee (% from sale) + $0.30 per order. Total ≈ **12.55% + $0.30** для most categories. Slightly cheaper than Amazon 15%.

---

## Fee structure

### 1. Insertion Fee

- **First 250 listings/month: FREE**
- After 250: $0.35 per listing

Vladimir's scope (~50-200 bundles initially) → likely под 250 limit → **$0 insertion fees**.

### 2. Final Value Fee (per sale)

| Category | FVF | Per-order fee |
|---|---|---|
| Most categories | 13.25% | $0.30 |
| Grocery / Food | **12.55%** | $0.30 |
| Pet Supplies | 13.25% | $0.30 |

### 3. Below threshold pricing

eBay charges higher FVF на portion of price above $7,500 (Higher Value Items). Not Vladimir's scope.

### 4. Promoted Listings (optional)

- Standard Promoted Listings: 2-12% ad rate
- Promoted Listings Advanced: CPC bidding

Phase 2 — start without promotion, scale based на conversion.

---

## Profitability example

Same $60 shelf-stable bundle:

| Item | Amount |
|---|---|
| Sale price | $60.00 |
| FVF (12.55%) | -$7.53 |
| Per-order fee | -$0.30 |
| Insertion (under 250) | -$0.00 |
| Components | -$18.00 |
| Packaging | -$2.00 |
| Shipping (FBM, ~5 lbs) | -$8.00 |
| **Net** | **$24.17** |

Comparable to Walmart ($24.80) и slightly better than Amazon ($23.00).

---

## eBay Store subscription (advanced)

For higher volume, eBay Store subscription:
- Basic: $4.95/month — extra free listings
- Premium: $21.95/month — more free + lower FVF в some categories
- Anchor: $59.95/month — significant savings

Vladimir's scope likely doesn't need subscription в Phase 2.
