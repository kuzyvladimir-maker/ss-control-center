# Amazon Category: Shelf-Stable Grocery

> **Source:** https://sellercentral.amazon.com/help/hub/reference/external/G201307470
> **Last verified:** 2026-05-17
> **Priority:** P1

---

## TL;DR

Shelf-stable (dry/canned) — самая простая категория. No cooler shipping. Ungating обычно не нужен (auto-approval для most subcategories). Это категория, которую Walmart Marketplace разрешает Vladimir-у.

---

## Hard rules

### 1. Storage temperature

`storage_temperature: Ambient` (или omit field).

### 2. Allergens, expiration

Те же что Frozen ([`category-frozen-grocery.md`](category-frozen-grocery.md)).

### 3. Approval

Большинство sub-categories — open auto-approval. Restricted: некоторые spices (saffron), certain teas, supplements.

### 4. Shipping

Standard shipping. No cooler. Veeqo Standard.

## Bundle examples

- Snack Variety Pack (chips + crackers + cookies — Cheez-Its, Pepperidge Farm, Pringles)

- Pasta Italian Night Kit (Barilla + Rao's sauce + Parmesan grater)
- Asian Pantry Set (soy sauce + ramen + rice + sriracha)
- Baking Essentials (flour + sugar + vanilla + chocolate chips)
