# Каталог бесплатных курсов ЛРХ (scientologycourses.org / Volunteer Ministers)

> 19 бесплатных онлайн-курсов «Инструменты для жизни» / Volunteer Ministers. Каждый курс = официальная бесплатная брошюра (PDF, церковь раздаёт свободно). Брошюры скачаны в `files/booklets/` (личная база Владимира). Курсы онлайн — бесплатно с регистрацией по email. Дословный текст курсов целиком не дублируется (копирайт) — но сами официальные брошюры с полным текстом лежат файлами.
